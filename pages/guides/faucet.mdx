# Testnet Faucets

Currently Chess.fish is on Mumbai and Celo Testnets. 


## Mumbai Faucet

To test the Chess.fish contracts, head to https://mumbaifaucet.com/ to get test Matic tokens. 

You can use https://faucet.polygon.technology/ to mint test ERC20 tokens.
